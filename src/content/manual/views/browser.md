## 2.1 When to use the Browser

Use the Browser when you want to find a sound, load a sample, open a preset, or add something to your project.

## 2.2 Opening the Browser

Tap the **Browser** button in the toolbar to open it. The Browser appears as a panel at the bottom of the screen.

## 2.3 Sections

### 2.3.1 Sounds

Browse built-in instrument presets organized by category:

- Keys
- Bass
- Pads
- Leads
- Drums
- Effects

Tap a preset to preview it. Tap **Load** to assign it to the current track.

### 2.3.2 Files

Navigate your device's file system to find audio files.

Supported formats:

- WAV
- OGG

### 2.3.3 Recent

Quick access to sounds and files you've used recently.

## 2.4 Search and preview

Use search to quickly find presets by name. Preview sounds before loading them — tap plays, tap again stops.

Drag a file onto a track to load it directly.
