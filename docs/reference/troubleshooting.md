# Troubleshooting

Common issues and how to resolve them.

## No sound

1. Check that your device volume is up and not in silent mode.
2. Make sure headphones or speakers are connected if expected.
3. Verify that the track has an instrument assigned.
4. Check that the track isn't muted in the Mixer.

## App won't start

1. Make sure your device meets the minimum requirements (Android 8+ / iOS 15+).
2. Try restarting the app.
3. If the problem persists, reinstall Pooly.

## Project won't load

1. Check that the project file hasn't been moved or renamed outside of Pooly.
2. Try opening a different project to rule out a general issue.

## Audio latency

1. Close other apps that may be using audio.
2. Use wired headphones instead of Bluetooth for lower latency.
3. Check the audio buffer size in Pooly's settings if available.
