# Install Pooly

!!! info "Early access"
    Pooly is currently in early development. Check the [Discord](https://discord.gg/zbNVJYBtw7) for the latest builds and platform availability.

## Requirements

- Android 8+ or iOS 15+
- Headphones or speakers recommended

## Installation

Installation instructions will be added here once Pooly is publicly available.

## First launch

When you open Pooly for the first time you'll see the **Project Selection** screen. From there you can create a new project or open an existing one.
