# Make Your First Song

This walkthrough takes you from an empty project to a short loop playing back.

## 1. Create a new project

Open Pooly and tap **New Project** on the Project Selection screen. Give it a name and tap **Create**.

## 2. Add a clip

You'll land in the **Clip Launcher**. Each cell in the grid represents a clip slot. Tap an empty slot to create a new clip.

## 3. Open the clip

Tap your new clip to open the **Clip Detail** view. This is where you draw or record notes.

## 4. Draw some notes

Use the **pencil tool** to tap notes onto the piano roll grid. Try placing a few notes on different beats.

## 5. Hit play

Tap the **play button** to hear your clip loop. Adjust notes, change the instrument, or add more clips to build up your song.
