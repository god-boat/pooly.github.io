# Clip Launcher

The **Clip Launcher** is Pooly's main performance and composition view. It presents a grid of clip slots organized by track (columns) and scene (rows).

## Overview

Each column is a track with its own instrument and effects. Each row is a scene. Tapping a clip slot lets you create, launch, or edit a clip.

## Creating clips

Tap an empty slot to create a new clip. The clip inherits the instrument assigned to that track.

## Launching clips

Tap a clip to launch it. It will start playing at the next quantization boundary (usually the next bar). Tap a running clip again to stop it.

## Launching scenes

Tap a **scene button** on the left edge to launch all clips in that row simultaneously. This is useful for switching between song sections.

!!! tip
    Use scenes to set up verse, chorus, and bridge sections, then trigger them during a live performance.
